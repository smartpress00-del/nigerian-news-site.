# Nigerian News Automation Project

## 📌 Overview
This project is designed to automatically collect Nigerian news headlines, rewrite them into more captivating versions, and publish them to a GitHub Pages website. The rewritten headlines are also shared to a connected Facebook Page using the Facebook Graph API. The goal is to build a professional, compliant system that drives traffic and enables monetization through Facebook and website ads.

---

## ⚙️ Features
- Collects Nigerian news headlines via RSS feeds or APIs.
- Rewrites headlines using natural language processing (NLP).
- Publishes updated headlines daily to a GitHub Pages website.
- Automatically posts rewritten headlines + links to Facebook.
- Designed to comply with Facebook Community Standards and Monetization Policies.

---

## 🛠 Requirements
- GitHub account and GitHub Pages enabled.
- Python 3.x installed (for headline rewriting and automation scripts).
- Facebook Developer Account with a Page Access Token.
- Optional: Zapier or IFTTT for no-code automation.
